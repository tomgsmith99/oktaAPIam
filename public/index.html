Okta is a standards-compliant OAuth 2.0 authorization server and a certified OpenID Provider.

The OAuth 2.0 API provides API security via scoped access tokens, and OpenID Connect provides user authentication and an SSO layer which is lighter and easier to use than SAML.

This site shows how Okta integrates with several leading API Management solutions.

A demo, setup guide, and architectural diagram is provided for each integration.
